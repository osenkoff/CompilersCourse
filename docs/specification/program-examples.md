# Примеры программ на StarLight

## 1. SumNumbers

Программа читает два числа и выводит их сумму.

Входные данные: Два числа (вводятся последовательно).

Выходные данные: Одно число - сумма.

```
ЗВЕЗДА
    СВЕТ a : нова;
    СВЕТ b : нова;
    СВЕТ сумма : нова;

    ПРИЕМ_СИГНАЛА(а);
    ПРИЕМ_СИГНАЛА(b);
    сумма = a + b;
    ИЗЛУЧАТЬ(сумма);
ЗАКРЫТАЯ_ЗВЕЗДА
```

## 2. FahrenheitToCelsius

Программа читает температуру в °F и выводит эквивалент в °C по формуле `c = (f - 32) * 5 / 9`.

Входные данные: Одно число - температура (°F).

Выходные данные: Одно число - температура (°C).

```
ЗВЕЗДА
    СВЕТ f : нова;
    СВЕТ c : нова;

    ПРИЕМ_СИГНАЛА(f);
    c = (f - 32) * 5 / 9;
    ИЗЛУЧАТЬ(c);
ЗАКРЫТАЯ_ЗВЕЗДА
```

## 3. QuadraticEquation

Программа читает коэффициенты a, b, c квадратного уравнения ax² + bx + c = 0 и выводит два корня по формуле `x = (-b ± sqrt(b² - 4ac)) / (2a)`. Предполагается дискриминант ≥ 0; sqrt как `** 0.5`.

Входные данные: Три числа — коэффициенты a, b, c (вводятся последовательно).
Выходные данные: Два числа — корни x1 (с +) и x2 (с -), разделенные пробелом.

```
ЗВЕЗДА
    СВЕТ a : нова;
    СВЕТ b : нова;
    СВЕТ c : нова;
    СВЕТ d : нова;
    СВЕТ sqrt_d : нова;
    СВЕТ x1 : нова;
    СВЕТ x2 : нова;

    ПРИЕМ_СИГНАЛА(a);
    ПРИЕМ_СИГНАЛА(b);
    ПРИЕМ_СИГНАЛА(c);

    d = b**2 - 4 * a * c;
    sqrt_d = d ** 0.5;
    x1 = (-b + sqrt_d) / (2 * a);
    x2 = (-b - sqrt_d) / (2 * a);
    ИЗЛУЧАТЬ(x1, x2);
ЗАКРЫТАЯ_ЗВЕЗДА
```

## 4. Factorial

 Читает число N и считает его факториал N!, используя цикл либо рекурсию.

 ```
 ЗВЕЗДА
    СВЕТ n : квазар;
    СВЕТ результат : квазар = 1;
    СВЕТ i : квазар;
    
    ФОТОН вычислить_факториал(число: квазар) : квазар {
        СВЕТ факториал : квазар = 1;
        СПЕКТР (i = 1; i <= число; i += 1) {
            факториал *= i;
        }
        ВЕРНУТЬ факториал;
    }
    
    ИЗЛУЧАТЬ("Введите число для вычисления факториала: ");
    ПРИЕМ_СИГНАЛА(n);
    
    ЕСЛИ (n < 0) {
        ИЗЛУЧАТЬ("Ошибка: факториал отрицательного числа не определен");
    } ИЛИ_НЕТ {
        результат = вычислить_факториал(n);
        ИЗЛУЧАТЬ("Факториал числа ", n, " равен ", результат);
    }
ЗАКРЫТАЯ_ЗВЕЗДА
```

## 6. MultiplicationTable 

Читает число N выводит таблицу умножения от 1×1 до N×N. Каждая i-я строка вывода — это результаты умножения числа i на числа от 1 до N.

```
ЗВЕЗДА
    СВЕТ n : квазар;
    СВЕТ i : квазар = 1;
    СВЕТ j : квазар = 1;

    ПРИЕМ_СИГНАЛА(n);

    СПЕКТР (i = 1; i <= n; i += 1) {
        СПЕКТР (j = 1; j <= n; j += 1) {
            ИЗЛУЧАТЬ(i, "*", j, "=", i * j);
        }
    }

ЗАКРЫТАЯ_ЗВЕЗДА;
```
