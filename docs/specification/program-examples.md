# Примеры программ на StarLight

## 1. SumNumbers

Программа читает два числа и выводит их сумму.

Входные данные: Два числа (вводятся последовательно).

Выходные данные: Одно число - сумма.

```
ЗВЕЗДА
    СВЕТ a : нова;
    СВЕТ b : нова;
    СВЕТ сумма : нова;

    ПРИЕМ_СИГНАЛА(а);
    ПРИЕМ_СИГНАЛА(b);
    сумма = a + b;
    ИЗЛУЧАТЬ(сумма);
ЗАКРЫТАЯ_ЗВЕЗДА
```

## 2. FahrenheitToCelsius

Программа читает температуру в °F и выводит эквивалент в °C по формуле `c = (f - 32) * 5 / 9`.

Входные данные: Одно число - температура (°F).

Выходные данные: Одно число - температура (°C).

```
ЗВЕЗДА
    СВЕТ f : нова;
    СВЕТ c : нова;

    ПРИЕМ_СИГНАЛА(f);
    c = (f - 32) * 5 / 9;
    ИЗЛУЧАТЬ(c);
ЗАКРЫТАЯ_ЗВЕЗДА
```

## 3. QuadraticEquation

Программа читает коэффициенты a, b, c квадратного уравнения ax² + bx + c = 0 и выводит два корня по формуле `x = (-b ± sqrt(b² - 4ac)) / (2a)`. Предполагается дискриминант ≥ 0; sqrt как `** 0.5`.

Входные данные: Три числа — коэффициенты a, b, c (вводятся последовательно).
Выходные данные: Два числа — корни x1 (с +) и x2 (с -), разделенные пробелом.

```
ЗВЕЗДА
    СВЕТ a : нова;
    СВЕТ b : нова;
    СВЕТ c : нова;
    СВЕТ d : нова;
    СВЕТ sqrt_d : нова;
    СВЕТ x1 : нова;
    СВЕТ x2 : нова;

    ПРИЕМ_СИГНАЛА(a);
    ПРИЕМ_СИГНАЛА(b);
    ПРИЕМ_СИГНАЛА(c);

    d = b**2 - 4 * a * c;
    sqrt_d = d ** 0.5;
    x1 = (-b + sqrt_d) / (2 * a);
    x2 = (-b - sqrt_d) / (2 * a);
    ИЗЛУЧАТЬ(x1, x2);
ЗАКРЫТАЯ_ЗВЕЗДА
```
