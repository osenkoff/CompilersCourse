# Список тестов синтаксического анализатора StarLight

## Базовые литералы

- [x] Разбор целого числа: `42`
- [x] Разбор десятичного числа: `3.14159`
- [x] Разбор логического значения ИСТИНА: `ИСТИНА`
- [x] Разбор логического значения ЛОЖЬ: `ЛОЖЬ`

## Идентификаторы и переменные

- [x] Разбор простого идентификатора: `переменная`
- [x] Разбор идентификатора с цифрами: `x1`
- [x] Разбор идентификатора на русском: `счетчик`

## Арифметические операции

- [x] Сложение: `2 + 3`
- [x] Вычитание: `5 - 2`
- [x] Умножение: `3 * 4`
- [x] Деление: `10 / 2`
- [x] Остаток от деления: `7 % 3`
- [x] Возведение в степень: `2 ** 3`
- [x] Комбинация операций: `2 + 3 * 4`
- [x] Приоритет операций: `(2 + 3) * 4`
- [x] Приоритет степени: `2 * 3 ** 4`
- [x] Сложные арифметические выражения: `2 + 3 * (4 - 1) ** 2`

## Унарные операции

- [x] Унарный плюс: `+5`
- [x] Унарный минус: `-5`
- [x] Логическое НЕ: `!ИСТИНА`
- [x] Приоритет унарного минуса: `-3 * 4`
- [x] Приоритет унарного с степенью: `-x ** 2`

## Операции сравнения

- [x] Равенство: `a == b`
- [x] Неравенство: `a != b`
- [x] Больше: `a > b`
- [x] Больше или равно: `a >= b`
- [x] Меньше: `a < b`
- [x] Меньше или равно: `a <= b`

## Логические операции

- [x] Логическое И: `a && b`
- [x] Логическое ИЛИ: `a || b`
- [x] Комбинация сравнений: `a > b && c < d`
- [x] Приоритет И над ИЛИ: `a && b || c && d`
- [x] Смешанные логические выражения: `a > 5 && b < 10 || c == 3`

## Присваивания

- [x] Простое присваивание: `x = 5`
- [x] Составное присваивание сложения: `x += 3`
- [x] Составное присваивание вычитания: `x -= 2`
- [x] Составное присваивание умножения: `x *= 4`
- [x] Составное присваивание деления: `x /= 2`
- [x] Составное присваивание степени: `x **= 2`

## Скобки

- [x] Выражение со скобками: `(2 + 3)`
- [x] Приоритет со скобками: `(2 + 3) * 4`
- [x] Вложенные скобки: `((2 + 3) * (4 - 1))`
- [x] Глубоко вложенные скобки: `((1 + (2 * (3 - (4 / 5)))))`

## Ассоциативность операторов

- [x] Левая ассоциативность сложения: `1 + 2 + 3`
- [x] Левая ассоциативность вычитания: `10 - 5 - 2`
- [x] Правая ассоциативность степени: `2 ** 3 ** 2`

## Сложные выражения

- [x] Смешанные типы операций: `a + b * c > d && e == f`

## Обработка ошибок

- [x] Неожиданный токен в выражении
- [x] Незакрытая скобка
- [x] Пустые скобки: `()`
- [x] Отсутствующий операнд: `2 +`
- [x] Неправильная последовательность операторов: `2 + * 3`
- [x] Неправильная последовательность операторов: `2 * * 3`
