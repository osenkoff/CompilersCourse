# Список тестов синтаксического анализатора StarLight

## Базовые литералы

- [x] Разбор целого числа: `42`
- [x] Разбор десятичного числа: `3.14159`
- [x] Разбор логического значения ИСТИНА: `ИСТИНА`
- [x] Разбор логического значения ЛОЖЬ: `ЛОЖЬ`

## Идентификаторы

- [x] Разбор простого идентификатора: `переменная`
- [x] Разбор идентификатора с цифрами: `x1`
- [x] Разбор идентификатора на русском языке: `счетчик`

## Арифметические операции

- [x] Сложение: `5 + 2 = 7`
- [x] Вычитание: `5 - 2 = 3`
- [x] Умножение: `3 * 4 = 12`
- [x] Деление: `10 / 2 = 5`
- [x] Остаток от деления: `7 % 3 = 1`
- [x] Возведение в степень: `2 ** 3 = 8`

## Ассоциативность

- [x] Правоассоциативная степень: `2 ** 3 ** 2 = 512`
- [x] Левоассоциативное сложение: `1 + 2 + 3 = 6`
- [x] Левоассоциативное вычитание: `10 - 5 - 2 = 3`

## Скобки и приоритет операций

- [x] Простое выражение в скобках: `(2 + 3) = 5`
- [x] Приоритет умножения над сложением: `2 + 3 * 4 = 14`
- [x] Явное изменение приоритета скобками: `(2 + 3) * 4 = 20`
- [x] Приоритет степени над умножением: `2 * 3 ** 4 = 162`
- [x] Вложенные скобки: `((2 + 3) * (4 - 1)) = 15`
- [x] Глубоко вложенное выражение: `((1 + (2 * (3 - (4 / 5))))) = 5.4`
- [x] Сложное арифметическое выражение: `2 + 3 * (4 - 1) ** 2 = 29`

## Унарные операции

- [x] Унарный минус: `-5`
- [x] Унарный плюс: `+5`
- [x] Логическое НЕ: `!ИСТИНА = ЛОЖЬ`
- [x] Приоритет унарного минуса над умножением: `-3 * 4 = -12`
- [x] Унарный минус и степень: `-x ** 2 = -9`
- [x] Унарные операции имеют меньший приоритет, чем степень: `-3 ** 2 = 9`

## Операции сравнения

- [x] Равенство (true): `5 == 5`
- [x] Равенство (false): `5 == 3`
- [x] Неравенство (true): `5 != 3`
- [x] Неравенство (false): `5 != 5`
- [x] Больше: `5 > 3`
- [x] Больше (false): `3 > 5`
- [x] Больше или равно: `5 >= 5`
- [x] Больше или равно (false): `3 >= 5`
- [x] Меньше: `3 < 5`
- [x] Меньше (false): `5 < 3`
- [x] Меньше или равно: `3 <= 5`
- [x] Меньше или равно (false): `5 <= 3`

## Логические операции

- [x] Логическое И: `ИСТИНА && ИСТИНА`
- [x] Логическое И (false): `ИСТИНА && ЛОЖЬ`
- [x] Логическое ИЛИ: `ИСТИНА || ЛОЖЬ`
- [x] Логическое ИЛИ (false): `ЛОЖЬ || ЛОЖЬ`
- [x] Проверка приоритета И над ИЛИ с вычислением: `(ИСТИНА && ЛОЖЬ) || (ИСТИНА && ИСТИНА)`

## Смешанные выражения

- [x] Сравнение + логика: `5 > 3 && 2 < 4`
- [x] Арифметика + сравнение + логика: `2 + 3 * 4 > 10 && 5 == 5`
- [x] Вычисление логического выражения с числами: `6 > 5 && 8 < 10 || 3 == 3`

## Присваивание

- [x] Простое присваивание: `x = 5`
- [x] Присваивание с плюсом: `x += 3`
- [x] Присваивание с минусом: `x -= 2`
- [x] Присваивание с умножением: `x *= 4`
- [x] Присваивание с делением: `x /= 2`
- [x] Присваивание со степенью: `x **= 2`

## Ошибки синтаксиса

- [x] Неожиданный конец выражения: `2 +`
- [x] Незакрытая скобка: `(2 + 3`
- [x] Пустые скобки: `()`
- [x] Отсутствует оператор: `2 + * 3`
- [x] Некорректная последовательность операторов: `2 * * 3`

## Инструкции ввода-вывода

- [x] Вывод числа: `ИЗЛУЧАТЬ(42);`
- [ ] Вывод выражения: `ИЗЛУЧАТЬ(2 + 3 * 4);`
- [ ] Вывод с несколькими аргументами: `ИЗЛУЧАТЬ(1, 2, 3);`
- [ ] Вывод переменной: `ИЗЛУЧАТЬ(x);`
- [ ] Ввод в переменную: `ПРИЕМ_СИГНАЛА(x);`

## Объявление переменных и констант

- [x] Объявление переменной с инициализацией: `СВЕТ x : квазар = 10;`
- [ ] Объявление переменной без инициализации: `СВЕТ y : нова;`
- [ ] Объявление константы: `КОНСТЕЛЛАЦИЯ пи : нова = 3.14159;`
- [ ] Объявление переменной типа вакуум: `СВЕТ флаг : вакуум = ИСТИНА;`

## Структура программы

- [ ] Минимальная программа: `ЗВЕЗДА ЗАКРЫТАЯ_ЗВЕЗДА`
- [ ] Программа с переменными: `ЗВЕЗДА СВЕТ x : квазар = 5; ЗАКРЫТАЯ_ЗВЕЗДА`
- [ ] Программа с несколькими инструкциями: `ЗВЕЗДА СВЕТ a : квазар = 1; СВЕТ b : квазар = 2; ИЗЛУЧАТЬ(a + b); ЗАКРЫТАЯ_ЗВЕЗДА`

## Присваивания

- [ ] Простое присваивание: `x = 42;`
- [ ] Составное присваивание с плюсом: `x += 5;`
- [ ] Составное присваивание с умножением: `x *= 2;`
- [ ] Присваивание с выражением: `y = x * 2 + 3;`

## Ошибки инструкций

- [ ] Отсутствует точка с запятой: `СВЕТ x : квазар = 5`
- [ ] Дублирующееся объявление: `СВЕТ x : квазар = 1; СВЕТ x : квазар = 2;`
- [ ] Использование необъявленной переменной: `x = 5;`
- [ ] Присваивание константе: `КОНСТЕЛЛАЦИЯ y : квазар = 5; y = 10;`
