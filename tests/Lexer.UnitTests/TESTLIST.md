# Список тестов

## Класс Lexer

- [x] Разбор ЗВЕЗДА ... ЗАКРЫТАЯ_ЗВЕЗДА

ЗВЕЗДА
    СВЕТ импульс : квазар = 42;
ЗАКРЫТАЯ_ЗВЕЗДА

- [x] Разбор СВЕТ (объявление переменной)

СВЕТ x : квазар = 1;

- [x] Разбор КОНСТЕЛЛАЦИЯ (объявление константы)

КОНСТЕЛЛАЦИЯ PI : нова = 3.1416;

- [x] Разбор ИЗЛУЧАТЬ

ИЗЛУЧАТЬ("Земля!");

- [x] ПРИЕМ_СИГНАЛА

СВЕТ имя = ПРИЕМ_СИГНАЛА();

- [x] Разбор ФОТОН и ВЕРНУТЬ

ФОТОН Сумма(a: квазар, b: квазар): квазар {
    ВЕРНУТЬ a + b;
}

- [x] Разбор ЕСЛИ без ИЛИ_НЕТ

ЕСЛИ (x > 0) { ИЗЛУЧАТЬ("Положительно"); }

- [x] Разбор цикла ОРБИТА

ОРБИТА (x < 10) {
    x += 1;
}

- [x] Разбор цикла СПЕКТР

СПЕКТР (i = 0; i < 10; i += 1) {
    ИЗЛУЧАТЬ(i);
}
